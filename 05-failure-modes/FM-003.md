# FM-003 — Знание без применения

**Описание:** материал изучен, но не изменил реальное поведение
**Причина:** нет явного шага «что я изменю после этого занятия»
**Сигнал:** много прочитано/прослушано, поведение не изменилось
**Как избежать:** после каждого занятия — один конкретный эксперимент с датой проверки результата
