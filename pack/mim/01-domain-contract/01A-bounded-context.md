## Критерии истинности

Утверждения в этом Pack проверяются:

1. **Наблюдаемость** — формат описан через наблюдаемые признаки (длительность, роли, рабочие продукты)
2. **Различимость** — каждый формат отличается от других по главному эффекту
3. **Тест на подмену** — если подменить один формат другим, результат изменится
4. **Воспроизводимость** — описание достаточно для организации мероприятия

## Ключевые различения контекста

| Различение | Почему критично |
|-----------|----------------|
| Формат ≠ Программа | Формат — тип мероприятия, программа — направленность развития. Смешение → неправильные ожидания |
| МИМ ≠ Сообщество | МИМ — организация (сотрудники, бюджет). Сообщество — объединение единомышленников |
| Мастерская ≠ Вуз | Бесконечное развитие с разборами vs конечное обучение с экзаменами |
| Развитие ≠ Обучение | Развитие бесконечно, через проблемы. Обучение конечно, через программу |

## Подсистема экосистемы

МИМ — одна из подсистем экосистемы развития интеллекта (PACK-ecosystem). Связь:

```
PACK-ecosystem (экосистема — чёрный ящик)
  ├── PACK-MIM ← этот Pack (организация развития)
  ├── PACK-digital-platform (ИТ-платформа)
  ├── PACK-personal (созидатель)
  └── [сообщество — пока в PACK-ecosystem]
```
