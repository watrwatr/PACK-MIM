# MIM.R.003 — Резидент

> Участник резидентуры: работает над своим «шедевром» под наставничеством.

## Summary

Резидент — участник резидентуры (MIM.FMT.003): специалист, который работает над своим рабочим результатом (шедевром, MIM.WP.002) при поддержке наставника. Проект свой, инициатива у резидента. Работает на основной работе, но проходит резидентуру, следуя регламентам организатора.

## Функции

1. **Выбирает проект** — свой шедевр (≠ стажировка, MIM.D.003)
2. **Работает над шедевром** (MIM.WP.002) — создаёт рабочий результат
3. **Приносит результат на разбор** — еженедельные review с наставником
4. **Следует регламентам** — сроки, критерии, промежуточные check-in

## Предпосылки

- Уже прошёл личное развитие (MIM.PRG.001) — владеет навыком
- Готов к самостоятельной работе над своим проектом
- Нет начальника, нет «часов работы» (≠ стажёр)

## Где действует

- **Резидентура** (MIM.FMT.003) в рамках программы рабочего развития (MIM.PRG.002)

## Чем отличается от стажёра (MIM.D.003)

| Резидент | Стажёр |
|----------|--------|
| Свой проект | Выданный проект |
| Инициатива у резидента | Инициатива у работодателя |
| Наставничество через review | Обучение на рабочем месте |
| Нет начальника | Есть начальник |
