# MIM.FMT.005 — Марафон

> Интенсивный композитный формат: фиксированная по времени последовательность из уроков, заданий и практик с ежедневным ритмом.

## Summary

Марафон — формат интенсивного развития, объединяющий базовые форматы (семинар, практикум, разбор) в фиксированную по времени последовательность с ежедневным ритмом. Отличается от отдельных форматов комбинированным эффектом: знакомство + тренировка + обратная связь в каждом цикле.

## Главный эффект

Интенсивное погружение + ритм + трекинг прогресса. Формирование привычки к ежедневной практике (deliberate practice).

## Наблюдаемые признаки

- Фиксированная длительность (типично: 14 дней)
- Ежедневный цикл: урок → задание → практика
- Комбинирует семинар (MIM.FMT.001) + практикум (MIM.FMT.002) + разбор (MIM.FMT.004)
- Трекинг прогресса: отставание, количество рабочих продуктов
- Рабочие продукты создаются ежедневно

## Рабочие продукты

- Рабочие продукты каждого дня (задания, практики)
- Замер квалификации (MIM.WP.003) — через вопросы и задания марафона

## Роли

- **Участник марафона** — проходит ежедневный цикл
- **Наставник** (MIM.R.001) — проверяет задания, даёт обратную связь (может быть автоматизирован через ИТ-платформу)

## Программа

Основной формат для вводного этапа **программы личного развития** (MIM.PRG.001). Может использоваться как точка входа в любую программу.

## Ключевые особенности

- Ритм задаёт дисциплину, а не содержание — содержание может быть из любой области
- Композитный: каждый день включает элементы нескольких базовых форматов
- Фиксированное время снижает барьер входа: участник знает, что это N дней, а не «бесконечно»
- Трекинг прогресса мотивирует через наблюдаемость (≠ мотивационные приёмы)

## Типичная ошибка

Марафон ≠ Курс. Курс — дидактическая конструкция (downstream). Марафон — организационный формат, задающий ритм и структуру ежедневной практики. Один и тот же контент может быть организован в формат марафона или нет.
