# MIM.FMT.002 — Практикум

> Формат тренировки навыка через осознанные упражнения (deliberate practice).

## Summary

Задача практикума — «набить руку» в навыке, чтобы базовые операции (в том числе мыслительные) перестали съедать внимание. Высвободившееся внимание направляется на адаптацию метода к разным ситуациям. Суть — не работа, а тренировка.

## Главный эффект

Автоматизация базовых операций, приобретение мастерства через deliberate practice (MIM.M.001).

## Наблюдаемые признаки

- Малые задания, повторения на границе возможностей
- Особое внимание к затратам времени и ритму (таймер)
- Регулярные разборы (MIM.FMT.004) с корректировками
- Безопасная среда, толерантная к ошибкам
- Нет обязательств кому-то, только перед собой

## Рабочие продукты

- Улучшенная версия себя (неотчуждаемый результат)
- «Учебный проект» — результат, который обычно выбрасывается за ненадобностью

## Роли

- **Наставник** (MIM.R.001) — ведёт разборы, даёт обратную связь
- Участник — выполняет упражнения
- **Интеллектуальный партнёр** (MIM.R.005, опционально) — горизонтальная проверка мышления между разборами (MIM.M.005)

## Программа

Основной формат **программы личного развития** (MIM.PRG.001).

## Ключевые особенности МИМ

- Аналоги: гаммы музыкантов, тренинг штриха у художников, тренировки спортсменов, лабораторные работы
- В разных организациях может называться: «тренинг», «челлендж», «учебный курс»
- Нацелен не на механические повторения, а на исключение ошибок по итогам обратной связи
- Подход adaptive expertise: повторяемость в основе + адаптация поверх
- Основная проблема: результат внутренний, предъявить можно только новое умение

## Чем отличается от резидентуры (MIM.FMT.003)

| Практикум | Резидентура |
|-----------|-------------|
| Освоение мастерства с нуля | Повышение квалификации в выдаче результата |
| Учебный результат (выбрасывается) | Рабочий результат («шедевр») |
| Начинающий | Уже профи |
| «Смотри, как я изменился» | Рабочие успехи |
