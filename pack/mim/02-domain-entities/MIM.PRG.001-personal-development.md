# MIM.PRG.001 — Программа личного развития

> Развитие самого инженера-менеджера: освоение мастерства и автоматизация базовых операций.

## Summary

Программа личного развития направлена на изменение самого человека: его навыков, мышления, способности действовать. Основной формат — практикум (MIM.FMT.002). Результат — «улучшенная версия себя», не рабочий продукт.

## Направленность

**Что развиваем:** самого инженера-менеджера — навыки, мышление, автоматизм базовых операций.

## Основной формат

**Практикум** (MIM.FMT.002) — тренировка навыка через deliberate practice (MIM.M.001).

## Результат

- Неотчуждаемый: «улучшенная версия себя»
- Предъявляемый: «смотри, как я изменился» (демонстрация навыка)
- Учебные проекты выбрасываются — ценность в изменении самого человека

## Обязательные элементы

- Регулярные практикумы с упражнениями
- Разборы (MIM.FMT.004) для корректировки
- Замер квалификации (MIM.WP.003) при каждом разборе
- Безопасная среда, толерантная к ошибкам

## Опциональные элементы

- Интеллектуальные партнёры (MIM.M.005) — парная проверка мышления между разборами

## Чем отличается от рабочего развития (MIM.PRG.002)

| Личное развитие | Рабочее развитие |
|-----------------|------------------|
| Практикум — тренировка | Резидентура — работа |
| Учебный результат (выбрасывается) | Рабочий результат (шедевр) |
| «Как я изменился» | «Что я сделал» |
| Навык как цель | Результат как цель |

## Типичная ошибка

Подмена личного развития рабочим: участник хочет «сразу делать полезное» вместо тренировки → пропускает этап освоения навыка → ошибки закрепляются (MIM.FM.002).
