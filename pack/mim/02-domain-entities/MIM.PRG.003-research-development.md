# MIM.PRG.003 — Программа исследовательского развития

> Развитие знания: знакомство с новым через семинары и коллективное понимание.

## Summary

Программа исследовательского развития направлена на расширение знаний — знакомство с новым и малодоступным материалом, который ещё не вошёл в руководства. Основной формат — семинар (MIM.FMT.001). Результат — коллективное понимание нового.

## Направленность

**Что развиваем:** знание — доступ к новому, сложному, малодоступному материалу.

## Основной формат

**Семинар** (MIM.FMT.001) — доклад и обсуждение нового знания.

## Результат

- Отчуждаемый: **слайдомент** (MIM.WP.001) — слайды-как-документ
- Неотчуждаемый: коллективное понимание нового материала
- Через время материал попадает в руководства и становится основой для практикумов

## Обязательные элементы

- Докладчик (MIM.R.002) с экспертизой в предмете
- Вопросы на понимание + проблематизацию
- Длительность: от пары часов до целого дня (6 часов)
- Слайдоменты как основной материал

## Место в цепочке развития

```
Исследовательское → Личное → Рабочее
(знакомство)        (навык)   (результат)
Семинар → Практикум → Резидентура
```

Семинар даёт знакомство с новым. Но послушать ≠ научиться (MIM.D.001). Для освоения навыка нужен практикум (MIM.PRG.001), для результата — резидентура (MIM.PRG.002).

## Ключевая особенность МИМ

Слайдомент можно сразу использовать с AI-агентом для обсуждения и пробной работы по новым методам — это мостик между «послушал» и «начал тренироваться».

## Типичная ошибка

«Послушал = научился» (MIM.FM.001): после семинара участник считает, что уже освоил материал → пропускает практикум → ошибки при попытке применить.
