# Pack Manifest: MIM

> Source-of-truth для предметной области «Мастерская инженеров-менеджеров» — организация и методики бесконечного профессионального развития.

---

## Scope

**В scope:**
- Форматы мероприятий развития (семинар, практикум, резидентура, разбор)
- Типы программ развития (личное, рабочее, исследовательское)
- Принципы организации развития (deliberate practice, adaptive expertise, бесконечное развитие)
- Роли организации (наставник, докладчик, резидент)
- Рабочие продукты форматов (слайдомент, шедевр, замер квалификации)
- Режимы ошибок в организации развития
- Различения между форматами и программами

**Вне scope:**
- Содержание конкретных курсов и руководств (downstream/surface)
- Характеристики и состояния созидателя (PACK-personal)
- ИТ-реализация платформы (PACK-digital-platform)
- Экосистема как целое (PACK-ecosystem)
- Конкретные расписания, потоки, группы (операционка)
- Мотивационные и рекламные тексты

### Различения (D)

| ID | Название | Summary |
|----|----------|---------|
| MIM.D.001 | Семинар ≠ Практикум | Знакомство с новым vs тренировка навыка. Послушать концерт ≠ научиться играть |
| MIM.D.002 | Практикум ≠ Резидентура | Тренировка (учебный результат) vs работа (рабочий «шедевр»). Гаммы ≠ концерт |
| MIM.D.003 | Резидентура ≠ Стажировка | Свой проект (сам выбираешь шедевр) vs выданные задания (начальник задаёт) |
| MIM.D.004 | Разбор ≠ Экзамен | Замер + обратная связь (непрерывно) vs оценка + допуск (в конце). Мастерская ≠ школа |
| MIM.D.005 | Формат ≠ Программа | Тип мероприятия (как устроено) vs направленность развития (что развиваем) |
| MIM.D.006 | Замещение функций ≠ Уничтожение рабочих мест | ИИ замещает операции, но создаёт новые роли. Путаница определяет образовательную стратегию |
| MIM.D.007 | Принцип ≠ Inductive Bias | Принцип — осознаваемое ограничение (S2). Inductive bias — встроенное в архитектуру ограничение (S1). Обучение = превращение принципов в biases через практику |
| MIM.D.008 | «Поумнеть» ≠ Повысить IQ | «Поумнеть» = решать новые классы задач (бесконечно). Повысить IQ = числовой показатель теста (ограничен биологией) |
| MIM.D.009 | Мастерская ≠ Школа | Мастерская = бесконечное становление мастерства (Mens et Manus). Школа = конечная программа с дипломом |
| MIM.D.010 | МИМ ≠ Сообщество | МИМ = организационная машина (сотрудники, бюджет, SoTA-research). Сообщество = объединение единомышленников |
| MIM.D.011 | Агентность ≠ Аналитическая установка | Агентность = инициирует интервенции, меняет мир. Аналитика = анализирует, оставляет действие другому. Умный ≠ агентный |
| MIM.D.012 | Мыслитель ≠ Созидатель (Делатель) | Написание текстов = мышление. Действие = выход в мир с ответственностью за риски. Обещания AI-агентов — речевые акты с реальными последствиями |
| MIM.D.013 | Критическая роль AI ≠ Генеративная роль AI | AI-критик (студент генерирует, AI оценивает) = deliberate practice + RLHF-аналог. AI-генератор (AI создаёт, человек редактирует) = риск атрофии компетенций |
