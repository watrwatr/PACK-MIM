# MIM.FM.002 — Заучивание ошибок

> Отсутствие разборов приводит к накоплению и закреплению незамеченных ошибок.

## Summary

Без регулярных разборов (MIM.FMT.004) участник повторяет неправильное и закрепляет ошибки. Deliberate practice (MIM.M.001) без обратной связи вырождается в механическое повторение. Результат — «выученная некомпетентность»: человек уверен в своём мастерстве, но делает неправильно.

## Как проявляется

- Участник тренируется много, но не улучшается
- Ошибки становятся «привычкой» — их трудно заметить самому
- Уверенность растёт, а качество — нет
- «Я уже опытный» + систематические ошибки

## Почему возникает

1. Отсутствие наставника (MIM.R.001) или редкие разборы
2. Самообучение без внешней обратной связи
3. Нетренированный глаз не видит собственных ошибок
4. Путаница: «много повторений = мастерство»

## Механизм

```
Практика без разбора
→ Ошибка не замечена
→ Повторение ошибки
→ Автоматизация ошибки
→ «Выученная некомпетентность»
```

## Как предотвратить

- Регулярные разборы: 1-2 раза в неделю (MIM.FMT.004)
- Наставник видит то, что автор не замечает (MIM.M.002)
- Обратная связь при каждой тренировке, а не только при «сдаче»
- Замер квалификации (MIM.WP.003) фиксирует прогресс (или его отсутствие)

## Принцип

> «Опытный наставник видит проблемы там, где нетренированный глаз не увидит.»
