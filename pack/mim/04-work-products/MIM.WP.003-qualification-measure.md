# MIM.WP.003 — Замер квалификации

> Результат разбора: оценка мастерства как индикатор прогресса, а не целевой показатель.

## Summary

Замер квалификации — рабочий продукт разбора (MIM.FMT.004): фиксация уровня мастерства участника при каждой встрече. Не экзамен и не целевой показатель, а индикатор продвижения. Квалификация замеряется непрерывно (при каждом разборе), а не одноразово (в конце курса).

## Ключевые свойства

- **Индикатор, не цель** — показывает прогресс, не является «оценкой для диплома»
- **Непрерывный** — при каждом разборе, а не «экзамен в конце»
- **Развивающий** — каждый замер — это одновременно обратная связь
- **Не разделяется** — нет отдельного «режима оценки» vs «режима обучения» (MIM.D.004)

## Чем замер НЕ является

| Замер квалификации | Экзамен |
|--------------------|---------|
| Индикатор прогресса | Допуск/оценка |
| При каждой встрече | В конце курса |
| Развитие + квалификация = одно | Обучение отдельно, оценка отдельно |
| Приносить сырое | Приносить «причёсанное» |
| Мастерская | Школа |

## Где создаётся

Разбор (MIM.FMT.004) — пронизывает все программы развития.

## Роль

Производит **наставник** (MIM.R.001) на основе review результатов.

## Почему важно

Снимает противоречие между «приносите сырой материал» и «а вот теперь экзамен». Когда замер — часть каждого разбора, участнику не нужно переключаться между режимами.
