# Онтология — PACK-mim

| Термин | Определение |
|--------|------------|
| Практика | Регулярное выполнение упражнения/метода с осознанным намерением |
| Дневник практики | Систематическая запись наблюдений до/после практики |
| Рефлексия | Осмысление опыта: что произошло, что изменилось, что применить |
| Ступень | Уровень мастерства в программе МИМ (1→5: Случайный→Проактивный) |
| Осознанность | Способность замечать собственные паттерны поведения и мышления в моменте |
| Неосознанность | Автоматическое действие без рефлексии — противоположность осознанности |
| Применение знаний | Изменение реального поведения на основе изученного — критерий освоения |
