# PACK-MIM

> **Тип репозитория:** `Pack`

Source-of-truth для предметной области **«Мастерская инженеров-менеджеров»** — организация и методики бесконечного профессионального развития.

---

## Что описывает этот Pack

МИМ (Мастерская инженеров-менеджеров) — организация, которая проектирует и ведёт программы развития инженеров-менеджеров. Этот Pack описывает **как устроено развитие**: форматы мероприятий, типы программ, принципы, роли организации, рабочие продукты и типичные ошибки.

### Форматы мероприятий (по главному эффекту)

| Формат | Главный эффект | Основная программа |
|--------|---------------|-------------------|
| **Семинар** | Знакомство с новым знанием | Исследовательское развитие |
| **Практикум** | Набивание руки (deliberate practice) | Личное развитие |
| **Резидентура** | Рабочий результат («шедевр») | Рабочее развитие |
| **Разбор** | Обратная связь + замер квалификации | Пронизывает все программы |

### Программы развития (по направленности)

| Программа | Объект развития | Основной формат |
|-----------|----------------|-----------------|
| Личное развитие | Сам инженер-менеджер | Практикум |
| Рабочее развитие | Результат на работе | Резидентура |
| Исследовательское развитие | Знание | Семинар |

---

## Pack ≠ курс

Этот Pack — **паспорт предметной области**, а не учебный материал:

| Pack (здесь) | Курс/руководство (downstream) |
|-------------|-------------------------------|
| Определения форматов и их различия | Пошаговые инструкции для участников |
| Принципы организации развития | Расписание и программа конкретного потока |
| Failure modes | Мотивационные тексты |

## Связь с другими Pack

- **PACK-personal (PD)** — описывает КТО развивается (созидатель). MIM описывает КАК организовано развитие
- **PACK-ecosystem (ECO)** — описывает экосистему как целое. MIM — одна из подсистем экосистемы
- **PACK-digital-platform (DP)** — ИТ-платформа. MIM описывает офлайн/онлайн форматы, платформа — техническую реализацию
